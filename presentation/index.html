User Notification

User Notification



  • Must register to use

  • Require user approval

User Notification





UIUserNotificationType types = UIUserNotificationTypeAlert |
UIUserNotificationTypeBadge | UIUserNotificationTypeSound;
UIUserNotificationSettings *settings =
[UIUserNotificationSettings settingsForTypes:types categories:nil];
[[UIApplication sharedApplication] registerUserNotificationSettings:settings];

User Notification


App Delegate Callback





- (void)application:(UIApplication *)application
didRegisterUserNotificationSettings:
(UIUserNotificationSettings *)notificationSettings
{
UIUserNotificationType allowedTypes = [notificationSettings types];
}

Remote Notification



  • User Type

    • Needs to register separately

  • Silent Type

    • Info.plist UIBackgroundModes array contains remote-notification

  • Can use both

Remote Notification



  • Must register before receiving pushes

  • Enabled by default

  • Can be disabled through settings

Remote Notification





UIApplication *myApp = [UIApplication sharedApplication];
[myapp registerForRemoteNotificationTypes:types];
[myapp registerForRemoteNotifications];
[myapp registerUserNotificationSettings:settings];

Notification Actions



  1. Register Actions and Categories

  2. Push/Schedule Notification

  3. Handle Actions

Register Actions and Categories



  • Actions appear as buttons

  • Group of actions is called category

  • UIMutableUserNotificationAction

  • UIMutableUserNotificationCategory



Push/Schedule Notification



  • "category" key for remote notif

  • "category" property for local notif

  • 2KB max size for remote notif



Handle Actions



  • - (void)application:(UIApplication *)application
    handleActionWithIdentifier:(NSString *)identifier
    forLocalNotification:(UILocalNotification *)notification
    forRemoteNotification:(UILocalNotification *)notification
    completionHandler:(void (^)())completionHandler;

  • completionHandler must be called
